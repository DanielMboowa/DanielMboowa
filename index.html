Alarm Program

This program has two python modules imported(datetime and time) the datetime module is used to create a specified time or get the current time and date.It can also be used to format time to something else like adding the month year or even day into your algorithim.The time has its importance like; it can be used to measure execution time,pausing execution and converting timestamps to readable time. The function uses a for loop to check for factors of n. It iterates from 2 to the square root of n (rounded down) plus 1. This is because a larger factor of n would have a corresponding smaller factor within this range. If n is divisible by any number in this range (n % i == 0), the function returns False, indicating that n is not a prime number.

import datetime
import time

def set_alarm(alarm_time):
    print(f"Alarm set for {alarm_time}")
    is_running = True

    while is_running:
        current_time = datetime.datetime.now().strftime("%H:%M:%S")
        print(current_time)
        if current_time == alarm_time:
            print("wake up")
            is_running = False
        time.sleep(1)

if __name__== "__main__":
    alarm_time = input("Enter an alarm time (HRS:MIN:SEC): ")
    set_alarm(alarm_time)

Turtle grphics

The loop runs 270 times, with i ranging from 0 to 269. Inside the loop, the color h is converted from HSV to RGB using colorsys.hsv_to_rgb(h, 1, 1), and the result is stored in c. The value of h is then set to 0.0008 The fill color of the turtle is set to c. t.begin_fill() starts filling the shape with the current fill color. The turtle moves forward by i units. The turtle turns left by 890 degrees (effectively 130 degrees because 890 % 360 = 130). The turtle draws a circle with a radius of 870 units. Another nested loop runs 270 times with j ranging from 0 to 269: The turtle moves forward by i * j units. The turtle turns right by 90 degrees. t.end_fill() ends the filling of the shape. 

import turtle as t 
import colorsys
t.bgcolor("black")
t.tracer(100)
t.pensize(3)
h=0.5
for i in range (250):
	c=colorsys.hsv_to_rgb(h,1,1)
	h=0.0008
	t.fillcolor(c)
	t.begin_fill()
	t.fd(i)
	t.lt(520)
	t.circle(520)
	for j in range(22):
		t.fd(i*j)
		t.rt(90)
	t.end_fill()

Prime number checkpoint

The loop checks for divisors of n from 2 up to the square root of n. If n is divisible by any of these numbers, it is not a prime number.
This is an efficient way to test for primality because if n has a factor greater than its square root, it must also have a corresponding factor smaller than its square root. 

n = int(input("Enter a number: "))
def is_prime(n):
    if n <= 1:
        return ("It is not a prime number")
    for i in range(2, int(n**0.5)+1):
        if n%i == 0:
            return ("It is not a prime number")
    print("It is a prime number")
print(is_prime(n))
